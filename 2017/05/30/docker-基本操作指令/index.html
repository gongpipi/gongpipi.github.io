docker 基本操作指令

2017-05-30
docker

安装docker

  1. yum install docker
  2. 启动服务 service docker start

推荐两个国内的docker镜像

docker 指令

  1. 拉取镜像
    docker pull redis
  2. 启动镜像
    docker run –name [name] -d [镜像id] -p [绑定到宿主机的端口]
    如 :
    docker run --name pipi-redis -d 0db2a690b89e -p 6379:6379
    docker run -d -p 6739:6739 0db2a690b89e
  3. 查看创建的容器
    如：docker ps -a
  4. 删除容器
    docker rm [容器id]
  5. 查看容器日志 ：
    docker logs [容器别名]
    如：docker logs pipi-redis
  6. 查看容器内部进程
    docker top [容器别名]
    如：docker top pipi-redis
  7. 获取容器更多的信息
    docker inspect [容器名]
    如：docker inspect pipi-redis
  8. 列举所有镜像
    如：docker images
  9. 查看对外端口
    docker port
    如： docker port
